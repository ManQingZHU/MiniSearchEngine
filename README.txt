Code for handling word stemming dowanloaded from the Internet. URL: https://tartarus.org/matrin/PorterStemmer/.
Stemming.cpp and Stemming.h are adjusted for better performance.

Use "make" command to get an executable file.

The file folder "documents" contains a "index.txt", which includes all the title of Shakespeare's work. You are expected to put text version of Shakespeare's work in here, so the program will fucntion correctly.

If your computer system is OS X, uncomment the code in main.cpp from line 89 - 97 before building.

If your computer system is Windows, use the code in main.cpp line 97.Also, you can use SearchEngine_win derictly.
